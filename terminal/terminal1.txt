 1. Создать файл “new.json” - touch new.json
 2. Добавить файл под гит. - git add new.json
 3. Закоммитить файл. - git commit -m "create new.json"
 4. Отправить файл на внешний GitHub репозиторий. - git push origin main
 5. Создать файлы new2.json, new2.xml, new2.txt - touch new2.json new2.xml new2.txt
 6. Добавить файлы под гит. - git add .
 7. Закоммитить файлы. - git commit -m "create new2.json, new2.txt, new2.xml"
 8. Отправить файлы на внешний GitHub репозиторий. - git push origin main
 9. Отредактировать содержание файла “new2.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). - vim new2.txt
 10. Отредактировать содержание файла “new2.json” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате JSON. - vim new2.json
 11. Добавить и закоммитить “new2.txt” одной строчкой. - git add new2.txt && commit -m "modified new2.txt"
 12. Добавить и закоммитить “new2.json” одной строчкой. - git add new2.json && commit -m "modified new2.json"
 13. Отправить изменения на внешний репозиторий. - git push origin main
 14. Отредактировать содержание файла “new2.xml” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате XML. - vim new2.xml
 15. Добавить и закоммитить “new2.xml” одной строчкой. - git commit -am "modified new2.xml"
 16. Отправить изменения на внешний репозиторий. - git push origin main
 17. Отправить предыдущее домашнее задание по GitBush Terminal на внешний репозиторий. + (в прошлом дз)
 18. В веб интерфейсе GitHub создать файл new3.txt. +
 19. Отредактировать в веб интерфейсе содержание файла “new3.txt” - написать информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить).+
 20. Сделать Commit changes (сохранить) изменения на веб интерфейсе. +
 21. Синхронизировать внешний и локальный репозитории (слить изменения с внешнего репозитория) - git pull origin main
 22. Отредактировать в веб интерфейсе содержание файла “new2.json” - добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить). Всё в формате JSON. +
 23. Сделать Commit changes (сохранить) изменения на веб интерфейсе. +
 24. Синхронизировать внешний и локальный репозитории - git pull origin main
 25. Отредактировать в веб интерфейсе содержание файла “new2.xml” - добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить). Всё в формате XML. +
 26. Сделать Commit changes (сохранить) изменения на веб интерфейсе. +
 27. Синхронизировать внешний и локальный репозитории - git pull origin main
